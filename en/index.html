Numalyse Player


The Numalyse Player is a software for media (video, image or sound) playback with advanced features.
It notably offers the display of media in the form of a playlist, the viewing of up to 4 media at the same time (with the possibility of synchronous playback) but also tools for working on the media (with among others: zoom, sequence extraction, drawing, annotation, facilitated shot segmentation...).

It is developed as part of the ANR Numalyse (2024-2028).

ℹ️ Questions

I own a MacBook. Do I have to do something else?

You must install VLC first : download the Universal Binary version.
⚠️ If you already have a version installed, the VLC "Universal Binary" version (link above) is yet highly recommended.

My MacBook told me that the application might be a malware, so I can't open it. What do I do?

Since our application is under an "alpha release", you will always have this prevention message from Apple.
To override temporily the Mac security settings : System Settings > Privacy and Security > Security > "NumalysePlayer" was blocked to protect your Mac [Open anyway]
(More information : Access Apple's user guide)

Which version of macOS can I use the software on?

The machines must have macOS 13 (Ventura) as a minimum configuration.

Why do I sometimes come across the term "SLV"?

This is the code name used to designate the software when it was created. Some artifacts from the application's old name may still remain.

How can I tell if I have the latest version of Numalyse Player?

In the software's "About" menu, a date is displayed. If this date does not match the one shown on this page, then the software is not the latest version. You can also contact us if you would like to receive email updates regarding future bug fixes.
