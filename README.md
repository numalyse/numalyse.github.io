# Numalyse Player

<div align="center">

<img src="https://raw.githubusercontent.com/numalyse/SLV/refs/heads/main/Ressources/logo/numalyse_player_logo.png" width="100" height="100">

</div>

<br>

<div align="center">

L'application *Numalyse Player* est un logiciel de lecture de médias (vidéo, image ou son) aux fonctionnalités avancées. <br>
Elle propose notamment l'affichage de médias sous forme d'une liste de lecture, le visionnage jusqu'à 4 médias à la fois (avec possibilité de lecture synchrone) mais également des outils de travail sur le média (avec entre autres : zoom, extraction de séquence, dessin, annotation, segmentation de plan facilitée...). <br><br>
Elle est développée dans le cadre de l'*[ANR Numalyse](https://anr.fr/Projet-ANR-24-CE54-7489)* (2024-2028).

</div>

## ℹ️ Foire aux questions

#### *Je suis propriétaire d'un MacBook. J'ai constaté qu'il y a deux versions. Laquelle dois-je installer ?*

> Vérifiez la puce de votre MacBook dans les informations générales de votre machine. [(Voir le guide d'installation)](https://upvdrive.univ-montp3.fr/s/83icE2KNKrZ9krj)


#### *Sous quelle version de macOS puis-je utiliser le logiciel ?*

> Les machines doivent avoir macOS 13 (Ventura) comme configuration minimale.


#### *Pourquoi je retrouve parfois le terme "SLV" ?*

> Il s'agit du nom de code utilisé pour désigner le logiciel lors de sa création. Il se peut que quelques artéfacts issus de l'ancien nom de l'application subsistent. 

#### *Comment savoir si j'ai la dernière version du Numalyse Player ?*

> Dans le menu "A propos" du logiciel, une date est indiquée. Si celle-ci ne correspond pas à celle indiquée sur cette page, alors le logiciel n'est pas dans sa dernière version. <br> Vous pouvez également nous contacter si vous souhaitez recevoir des mails informatifs concernant les futures mises à jours correctives.
